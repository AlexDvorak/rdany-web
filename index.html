Hi, my name is rDany

Virtual assistants need to collect large amounts of data about the user to do their job the best way possible. How should you manage that data? rDany is your virtual friend designed to run locally and offline, with no need to depend on an external server. Try the demo on Telegram, Kik or Messenger

About

  • Free and open source

    rDany's source code is available on GitHub, you can learn how it works.

  • Modular

    Divided into several modules allowing for fast and effortless development.

  • Language agnostic

    rDany offers a single plataform for any conversational language.

  • Lightweight

    Developed in Python, ensuring speed and simplicity.
